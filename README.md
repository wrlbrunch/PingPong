# Pong - CS50 Games

Este projeto é uma releitura do clássico **Pong**, desenvolvida como o primeiro desafio do curso **CS50's Introduction to Game Development**. O foco foi aplicar conceitos fundamentais de desenvolvimento de jogos, como detecção de colisão, gerenciamento de estados e inteligência artificial.

## Tecnologias
* **Linguagem:** Lua
* **Engine:** [LÖVE2D](https://love2d.org/)
* **Biblioteca:** Push (para manter a resolução virtual retro)

## Como Jogar
Para rodar este jogo, você precisa ter o **LÖVE2D** instalado em seu sistema.

1. **Baixe o LÖVE:** [love2d.org](https://love2d.org/)
2. **Execute o projeto:** No terminal, dentro da pasta do jogo, digite:
   ```bash
   love .
3.  **Execute o jogo:**
    Navegue até a pasta do projeto e arraste a pasta para o executável do LÖVE ou use o terminal:
    ```bash
    love .
    ```

## Controles
* **Jogador 1 (Esquerda):** Teclas `W` e `S`
* **Jogador 2 (Direita):** Setas `Cima` e `Baixo`
* **Iniciar:** Tecla `Enter`

## O que eu aprendi:
* Manipulação de coordenadas 2D ($x, y$).
* Detecção de colisão (AABB).
* Gerenciamento de estados (Menu, Play, Serve, Victory).
* Implementação de efeitos sonoros e fontes customizadas.
